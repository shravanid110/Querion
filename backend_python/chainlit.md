# Welcome to Querion 🚀

Querion is your intelligent companion for database analysis and backend monitoring.

### 📊 Database Mode
Ask questions in natural language, and Querion will generate and execute SQL queries on your connected databases.

### 🔍 Backend Monitoring
Switch to monitoring mode to watch your project's code and logs in real-time. Querion will identify issues, explain root causes, and suggest fixes with full context.

---
**Get started by connecting a project:**
`pip install querion-cli`
`querion watch --project "MyProject"`
